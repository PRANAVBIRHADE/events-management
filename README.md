# Freshers Party Event Site

## Setup

1. Clone the repo
2. Run `npm install`
3. Add your Supabase credentials to `.env.local`
4. Run `npm start`

## Features
- User registration/login
- Event listing & registration
- Admin dashboard (protected)
- Chakra UI styling
- Supabase backend

## Supabase Setup
- Create tables using the provided SQL
- Add your admin user

## Tech Stack
- React + TypeScript
- Chakra UI
- Supabase
- React Router
